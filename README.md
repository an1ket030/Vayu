# CleanAir

AI-powered air quality intelligence platform focused on India (Delhi NCR).

## Tech Stack
- **Frontend:** React 18, TypeScript, Vite, Tailwind CSS, shadcn/ui
- **Backend:** Node.js, Express, TypeScript, Prisma, PostgreSQL
- **ML/Python:** FastAPI, XGBoost
- **Cache/Queue:** Redis, BullMQ
- **Maps:** Leaflet.js

## Features
- Real-time AQI Monitoring
- ML-based 24h & 7-day AQI Prediction
- Clean-Air Route Finder
- Personal Exposure Tracker

## Local Setup ###Aniket
1. Clone the repository.
2. Copy `.env.example` to `.env` and fill in the API keys.
3. Run `docker-compose up`.
